Welcome to WP Socializer V3

WP Socializer is back again with completely new features and functionality to enrich your WordPress site with social sharing buttons.

This new version is redesigned from scratch with new administration UI and an vision to become an all in one social sharing plugin for WordPress.

Features

  • New easy drag & drop Administration UI
  • Native social sharing buttons
  • Floating sharebars
  • Share counters
  • Follow Us or profile media icons
  • Mobile sharebar
  • Selected text sharebar
  • Conditionally display social sharing buttons in posts and pages
  • Ultimate flexibility to customize the social buttons.
